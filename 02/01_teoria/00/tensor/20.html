Tensor

Tensores é a forma como definimos as grandezas, que servem para medir a realizade.

Se quisermos medir a temperatura, por exemplo, pode usar um número, como 20 °C. O valor é a magnitute e o °C é a unidade da medida, ela diz como você mediu o valor 20. Com outro termômetro teria outro valor.

Quando só a magnitude é suficiente, chamamos de tensor rank 0, ou escalar.

Mas digamos que vamos medir a velocidade de um carro. O carro pode ter a velocidade de 20 m/s. De novo temos a magnitude de 20, mas a unidade de m/s. Porém, um carro que se move para a direita é diferente do que se move para a esquerda. Para medir a velocidade corretamente, precisamos definir o espaço no qual esse móvel de andou.

Com isso surge valores quando a direção e sentido dessa grandeza. Grandezas desse tipo são tensores de rank 1, ou chamado de vetores.

Agora precisamos de uma matriz para conseguir descrever as componentes dessa grandeza, dessa forma, essas grandezas são tensores de rank 2, também chamado de tensores.

Por fim, você deve imaginar que não para por aí. Os ranks de tensores seão maiores quanto maior for a "dimensão" das componentes necessárias para descrever a grandeza.
